## Recent Projects

### Boring Festival 2026

**Tags:** Web development / Statamic
**URL:** https://www.boringfestival.nl/

Boring Festival is a music, theatre and art festival in Haarlem, taking place on February 6th and 7th, 2026.

Based on the awesome design by [Arthur Reinders Folmer](https://www.arthus.nl/), I built a multilingual Statamic site with dynamic features like act filtering and a live display system for the current and upcoming performances.

### Quickhost

**Tags:** Web design / Web development / Statamic
**URL:** https://www.quickhost.nl/

Quickhost is a web hosting company from Groningen, NL.

I designed and built their new website featuring a knowledge base where customers can easily search and browse tutorials by category.

I also restyled their order system and connected it to the main site, so menus stay in sync and everything feels like one seamless experience.

### Villa del Valle

**Tags:** Web design / Web development
**URL:** https://www.villadelvallecallosa.com/

Villa del Valle needed a website that would attract international guests to their beautiful villa on the Costa Blanca.

Working from their graphic designer's brand guidelines, I created a multilingual WordPress site with seamless booking integration.

### World Nautic Center

**Tags:** UX consultancy / Web development / Design system
**URL:** https://www.worldnauticcenter.nl/

World Nautic Center is a leading watersports equipment retailer in the Netherlands.

After doing a UX review, I built a new front-end system that improved design consistency and user experience across their 10+ websites.

### Kleinstesoepfabriek

**Tags:** Web development / E-commerce
**URL:** https://kleinstesoepfabriek.nl/

Kleinstesoepfabriek is a soup company in Leek, NL that makes delicious soups from organic and fairtrade ingredients.

I built their e-commerce site with extensive WooCommerce customization.

Design by BrandMonks & ConversieKracht.

### Conversiekracht

**Tags:** Web development / WordPress
**URL:** https://conversiekracht.nl

Conversiekracht is a conversion rate optimization agency from Groningen, NL that helps businesses turn more visitors into customers.

I built their new WordPress site with some cool interactive features, my favorite being a revenue calculator that helps visitors see exactly how much extra revenue they could generate.

Design by De Designpraktijk.

### FODM.app

**Tags:** Web development / Laravel / Livewire
**URL:** https://fodm.app

FODM.app is a recipe manager for people who follow the FODMAP diet, like myself. It also works for managing histamine intolerance, diabetes-friendly meals, food allergies, and more!

This app is built in Laravel and Livewire.

### LEQR menu

**Tags:** Web design / Web development / Laravel
**URL:** https://leqrmenu.com

LEQR menu is a digital menu system for restaurants. It handles real-time availability updates, dynamic pricing, filtering by dietary preferences, and other interactive features that are not possible with traditional paper menus.

The initial Laravel app was developed by [Laurens Kuiper](https://www.laurenskuiper.com/). I designed and built the new landing page, and we're now working on exciting new features to help restaurants serve their customers better.

## About Me

### More about me

Back in 2012 I built my first website for my driving instructor. Terrible code, but it worked, and I got 10 free driving lessons out of it. Seeing it help his business felt amazing. That feeling stuck with me.

These days, I'm selective about the projects I take on. I want to work with organizations doing meaningful work: creative projects, social impact, things that actually help people or make the world better.

I'll ask the right questions, challenge assumptions when needed, and suggest improvements you might not have considered. My best clients love this collaborative approach.

I pay close attention to the details that make a website feel polished and easy to use: smooth micro interactions, thoughtful flow, clean design. The web should work for people, not frustrate them.

If you're doing work that matters, let's talk.
